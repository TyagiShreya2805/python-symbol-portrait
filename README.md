# Python Symbol Portrait

A computer vision project that transforms portrait photographs into artistic portraits made entirely from Python syntax symbols.

---

## Features

- Load portrait images
- Convert images to grayscale
- Face detection *(coming soon)*
- Background removal *(coming soon)*
- Python syntax symbol rendering *(coming soon)*

---

## Tech Stack

- Python
- OpenCV
- NumPy
- Pillow
